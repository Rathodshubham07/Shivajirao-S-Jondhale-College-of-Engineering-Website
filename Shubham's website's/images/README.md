# Images Folder

Place all your image assets in this folder.

## Recommended Images:

- Logo files (logo.png, logo.svg)
- Portfolio project images
- Profile photos
- Background images
- Any other graphics used on your website

## Image Formats:

- **PNG** - For images with transparency
- **JPG/JPEG** - For photographs
- **SVG** - For scalable graphics and icons
- **WebP** - Modern format with better compression

## File Naming:

Use descriptive, lowercase names with hyphens:
- `portfolio-project-1.jpg`
- `hero-background.png`
- `profile-photo.jpg`
