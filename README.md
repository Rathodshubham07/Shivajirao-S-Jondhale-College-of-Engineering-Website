## Setup Instructions

### 1. Clone or Download this Repository
```bash
git clone <your-repo-url>
cd <your-repo-name>
```

### 2. Customize Your Content

- Edit `index.html` and other HTML files to replace placeholder content
- Update contact information in `contact.html`
- Replace placeholder images in `portfolio.html` with your actual work
- Modify `style.css` to change colors, fonts, and styling
- Add your own `favicon.ico` file

### 3. Deploy to GitHub Pages

1. Push your code to a GitHub repository
2. Go to your repository settings
3. Navigate to "Pages" section
4. Select your main branch (usually `main` or `master`)
5. Save - Your site will be live at `https://<username>.github.io/<repository-name>`

## Customization Guide

### Changing Colors

Edit the CSS variables in `style.css`:

```css
:root {
    --primary-color: #4a90e2;
    --secondary-color: #2c3e50;
    --accent-color: #e74c3c;
    /* ... */
}
```

### Adding Your Logo

Replace the logo text in the navigation bar or add an image:

```html
<div class="logo">
    <img src="images/logo.png" alt="Your Logo">
</div>
```

### Updating Contact Information

Edit `contact.html` and replace:
- Email address
- Phone number
- Location
